## Git shortcuts
https://github.com/robbyrussell/oh-my-zsh/tree/master/plugins/git/

| Command                    | Shortcut |
| -------------------------- | -------- |
| git add --all              | **gaa**      |
| git commit -v              | **gc**       |
| git checkout -b            | **gcb**      |
| git checkout master        | **gcm**      |
| git checkout               | **gco**      |
| git fetch --all --prune    | **gfa**      |
| git commit --amend         | **gc!**      |
| git rebase -i              | **grbi**     |
| git status                 | **gst**      |
| git pull --rebase          | **gup**      |
